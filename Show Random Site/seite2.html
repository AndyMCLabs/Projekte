Einführung in CSS

CSS steht für Cascading Style Sheets und wird verwendet, um das Aussehen und das Layout einer Webseite zu gestalten. Mit CSS kannst du Farben, Schriftarten, Abstände und andere Stilmerkmale definieren.

Grundlegende CSS-Syntax

CSS verwendet Selektoren, um HTML-Elemente zu bestimmen, die gestylt werden sollen. Die grundlegende Syntax sieht so aus:

selector {
    property: value;
}

Ein einfaches Beispiel

Hier ist ein Beispiel für CSS, das auf eine HTML-Seite angewendet wird:

/* Dies ist ein Kommentar */
body {
    background-color: #f0f0f0;
    color: #333;
    font-family: Arial, sans-serif;
}

h1 {
    color: #0066cc;
}

p {
    font-size: 16px;
}

Dieses Beispiel zeigt, wie man den Hintergrund der Seite, die Schriftfarbe von Überschriften und die Schriftgröße von Absätzen ändern kann.
