Einführung in HTML

HTML steht für Hypertext Markup Language und ist die grundlegende Sprache, die verwendet wird, um Webseiten zu erstellen. HTML wird verwendet, um die Struktur und den Inhalt einer Webseite zu definieren.

Grundlegende HTML-Struktur

Eine einfache HTML-Seite besteht aus den folgenden Hauptteilen:

  • <!DOCTYPE html>: Definiert den Dokumenttyp und die HTML-Version.
  • <html>: Das Wurzel-Element einer HTML-Seite.
  • <head>: Enthält Metadaten über das Dokument, wie den Titel.
  • <body>: Beinhaltet den sichtbaren Inhalt der Webseite.

Ein einfaches Beispiel

    <!DOCTYPE html>
    <html lang="de">
    <head>
        <meta charset="UTF-8">
        <title>Meine erste Webseite</title>
    </head>
    <body>
        <h1>Hallo Welt!</h1>
        <p>Dies ist mein erster Absatz.</p>
    </body>
    </html>

Dieses Beispiel zeigt eine sehr grundlegende HTML-Seite mit einem Titel und einem einfachen Textabschnitt.
