Einführung in JavaScript

JavaScript ist eine Skriptsprache, die verwendet wird, um Webseiten interaktiv zu machen. Mit JavaScript kannst du Benutzereingaben verarbeiten, Inhalte dynamisch ändern und viele andere interaktive Funktionen implementieren.

Grundlegende JavaScript-Syntax

JavaScript wird häufig in HTML-Dokumenten innerhalb von <script>-Tags eingebettet. Hier ist ein einfaches Beispiel:

<script>
    // Dies ist ein Kommentar
    alert('Hallo, Welt!');
</script>

Ein einfaches Beispiel

Hier ist ein Beispiel für ein JavaScript-Skript, das auf einen Button-Klick reagiert:

<!DOCTYPE html>
<html lang="de">
<head>
    <meta charset="UTF-8">
    <title>JavaScript Beispiel</title>
</head>
<body>
    <button onclick="zeigeNachricht()">Klicke mich</button>

    <script>
        function zeigeNachricht() {
            alert('Du hast den Button geklickt!');
        }
    </script>
</body>
</html>

Dieses Beispiel zeigt, wie du eine Funktion definierst, die eine Nachricht anzeigt, wenn der Button auf der Seite geklickt wird.
